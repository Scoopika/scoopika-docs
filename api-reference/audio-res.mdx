---
title: AudioRes
---

Represents an audio chunk generated by the agent during a run (using voice).

## Props

<ParamField path="index" type="number">
    The index of the audio chunk.
</ParamField>

<ParamField path="read" type="string">
    A link to read the generated audio. it can only be used once.
</ParamField>

<ParamField path="run_id" type="string">
    The ID of the run this audio was generated during.
</ParamField>

<ParamField path="audio_id" type="string">
    The ID of the audio chunk.
</ParamField>

<ResponseExample>
```json Example
{
    "index": 0,
    "read": "path-to-audio",
    "run_id": "run_1",
    "audio_id": "audio_123"
}
```
</ResponseExample>
