title: Initialize Agent

An `Agent` instance enables you to interact with individual agents. [What are agents](/agents)

## Arguments

<ParamField path="id" type="string" required>
  The agent ID. your agent ID is shown under its name in the [platform](https://scoopika.com/app), just click it to copy.
</ParamField>

<ParamField path="client" type="Client" required>
  An instance of the Scoopika Client class that you previously initialized.

  [Reference to Client instance](/packages/ts/client/client)
</ParamField>

<RequestExample>
```typescript Usage example
import { Client, Agent } from "@scoopika/client";

const client = new Client("API_URL"); // Replace with your Scoopika API route
const agent = new Agent("AGENT_ID", client);
```
</RequestExample>
