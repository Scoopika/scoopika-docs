---
title: Initialize Client
---

To get started, you need to initialize a `Client` instance that's passed to any new agent or box instance you initialize later, and used to manage chat sessions.

## Arguments

<ParamField path="url" type="string" required>
  The Url to your server-side API route that uses your Scoopika Endpoint, if following the docs, it should be under the `/scoopika` route. Learn more about how to setup Scoopika in your web application [here](/guides/scoopika-for-the-web).
</ParamField>

<ResponseExample>
```typescript Usage
import { Client } from "@scoopika/client";

const client = new Client("API_URL");
```
</ResponseExample>
