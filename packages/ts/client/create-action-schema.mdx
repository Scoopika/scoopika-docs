---
title: createActionSchema
---

Used to create a [client-side action](/tools/client-side-actions) schema from a JSON schema.

## Arguments

Takes one arguments passed an object with the following properties:

<ParamField path="name" type="string" required>
	The name of the tool or action.
</ParamField>

<ParamField path="description" type="string" required>
	The description of the tool or action. keep your description clear as it guides the agent when to use the tool.
</ParamField>

<ParamField path="parameters" type="JSONSchema" required>
	The parameters schema of the action's function. recommended to be a `JSONSchema` as `object` in the parent property. this schema is used to validate the arguments passed from the agent to your action to ensure stability.

	[Reference to JSONSchema](https://json-schema.org/learn/getting-started-step-by-step)
</ParamField>

## Return Value

<ResponseField name="tool_schema" type="ToolSchema">
	Returns a tool schema that can be then used to inject the too into agents (using [addClientAction](/packages/ts/client/agents/methods/add-client-action)) and boxes.

	[Reference to ToolSchema](/api-reference/tool-schema)
</ResponseField>

<ResponseExample>
```typescript Usage
import { createActionSchema } from "@scoopika/client";

const toolSchema = createActionSchema({
	name: "sort_documents",
	description: "Sort the user documents from latest to oldest",
	parameters: {
		type: "object",
		properties: {
			field: {
				type: "string",
				description: "The field to sort the documents by",
				enum: ["title", "date"]
			}
		},
		required: ["field"]
	}
});
```

```typescript Type-safe usage
import {
	createActionSchema,
	FromSchema,
	JSONSchema
} from "@scoopika/client";

// Define the tool parameters schema
const toolParams = {
	type: "object",
	properties: {
		id: {
			type: "string",
			description: "The document ID to get info about",
		}
	},
	required: ["id"]
} as const satisfies JSONSchema;

// Create a type for the tool's parameters or inputs
type ToolInputs = FromSchema<typeof toolParams>;

// Create the tool schema
const toolSchema = createActionSchema({
	name: "get_document_info",
	description: "Get information about a document",
	parameters: toolParams
});
```
</ResponseExample>
