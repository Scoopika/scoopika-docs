## Initialization

Before interacting with agents and boxes, you'll need to initialize a Scoopika instance. This instance acts as a central hub for managing your Scoopika interactions, including functionalities like history retrieval.

1. Import the Scoopika class from the package.
2. Create a new Scoopika instance, providing the following configuration options:
	- token: Your unique Scoopika token. [generate token](https://scoopika.com/app/settings?tab=tokens).
	- engines: An object containing API keys for the LLM providers your agents utilize (refer to the [engines documentation](/packages/ts/scoopika/scoopika) for details).

```typescript
import { Scoopika } from '@scoopika/scoopika';

const scoopika = new Scoopika({
	token: 'YOUR_SCOOPIKA_TOKEN',
	engines: {
		openai: 'OPENAI_KEY' // replace with your key
	}
});
```

Notice that in the engines, you need to provide keys for the providers your agents use.

Check all options you can pass to a Scoopika instance [here](/packages/ts/scoopika/scoopika).

## Usage Example

```typescript
import { Scoopika, Agent, Box } from '@scoopika/scoopika';

const scoopika = new Scoopika({
	token: 'YOUR_SCOOPIKA_TOKEN',
	engines: {
		openai: 'OPENAI_KEY'
	}
});

const agent = new Agent('AGENT_ID', scoopika);

(async () => {
	// Create session for user USER_1
	const session = await scoopika.newSession({
		user_id: "USER_1"
	});
	console.log(session); // StoreSession

	// Run agent
	const response = await agent.run({
		inputs: {
			message: 'Hello!',
			session_id: session.id
		},
		hooks: { // streaming hooks
			onToken: (t) => console.log(t)
		}
	});
	console.log(response); // AgentResponse

	// List user sessions
	const userSessions = await scoopika.listUserSessions("USER_1");
	console.log(userSessions); // string[]

	// Get session runs (history or messages)
	const runs = await scoopika.getSessionRuns(userSessions[0]);
	console.log(runs); // RunHistory[] (in this example, there're 2 items one for user request and one for agent response)
})();
```
