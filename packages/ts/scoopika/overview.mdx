---
title: 'Overview'
description: 'The core package for running Scoopika on the server-side'
---

This core package equips you to run Scoopika functionalities directly on your server-side environment. It caters to developers seeking granular control and customization beyond the simplified server setup offered by `@scoopika/server`.

Links:

[<Icon icon="npm" /> npm](https://npmjs.com/package/@scoopika/scoopika) |
[<Icon icon="github" /> Github](https://github.com/Scoopika/scoopika)

---

This package is designed for:

- Running agents.
- Running smart boxes.
- Adding external tools to agents and boxes.
- Connecting to remote history data stores and managing sessions.
- Extracting structured output using agents.
